Motores del proyecto Forseti
¿Que es el software libre y por que usarlo ?

¿Qué es el software libre?

Es el que te permite:

1. La libertad de uso en cualquier equipo y para cualquier fin.
2. La libertad de copiarlo y/o reproducirlo para cualquier otra persona o empresa.
3. La libertad de estudiar su código fuente; es decir, conocer la receta de ese guiso.
4. La libertad para adaptar ese programa a tus necesidades.

¿Por qué dar el salto?

La situación económica actual, en la que tu empresa es muy sensible al aumento de costos, y en donde las licencias de software te desfavorecen cada vez más, hace del software libre una alternativa muy atractiva.

¿Qué ventajas tiene el uso del software libre para mi empresa?

Hay muchas ventajas, algunas por ejemplo:
  • Las licencias no cuestan nada.
  • Se puede adaptar a tus necesidades (porque el código es libre y abierto).
  • No tienes que esperar a actualizaciones de software, porque el código abierto está en continua revisión y todas sus versiones están disponibles en tiempo real.
  • Tiene una alta estabilidad (porque la gran mayoría está basado en Linux).
  • Mayores posibilidades de control.
¿Qué calidad tiene el software libre?

Se considera en general de mayor fiabilidad que los de cualquier otro sistema cerrado, ya que los mecanismos de control y revisión son mucho más exigentes. La forma de trabajar de los desarrolladores es equivalente al modo de trabajo científico de universidades.

¿Es seguro el software libre?

  • El código fuente puede ser visto por muchas personas, por ello pueden detectarse y corregirse rápidamente posibles errores y cadencias que posee el sistema de seguridad.
  • La búsqueda de errores sólo puede llevarse a cabo en el código fuente: este último sólo está disponible si el software es libre.
  • Los códigos, al estar abiertos a la humanidad, pueden perfeccionarse rápidamente.
  • La mayor parte de los proyectos se basan en el sistema operativo Linux y como se viene mencionando cada vez más en los últimos años, Linux es el sistema más seguro que existe actualmente.
¿Quién apoya o está instalando software libre?

Ha llegado a convertirse en una alternativa conocida y muy atractiva para ofertas comerciales, del que la Unión Europea, América del Norte y un gran número de Administraciones y empresas, se han podido beneficiar. Grandes firmas como IBM, Hewlett Packard o Google Inc. son potenciadores de sistemas de código abierto y de sus desarrollos. Asimismo importantes fabricantes de software comercial como Oracle o SAP tienen muchos puntos en común con el software libre.
