Motores del proyecto Forseti
¿Es seguro Forseti ?
 POR SUPUESTO... Forseti está diseñado para la nube y utiliza motores sumamente poderosos:
  • Forseti utiliza PostgreSQL, una base de datos probada, que se ha ganado una sólida reputación de fiabilidad e integridad de datos, que maneja perfectamente múltiples transacciones simultaneas.

  • Forseti utiliza Apache Tomcat, el estándar de servidores web basados en Java, este controla perfectamente las sesiones con los distintos navegadores, y la comunicación sobre el protocolo https ( Protocolo Seguro de Transferencia de Hipertexto ). Bajo este protocolo, trabajan los portales bancarios y las tiendas en línea ya que se encuentran diseñados para la transmisión de datos cifrados.
Otro factor muy importante para la seguridad es que Forseti separa el control físico del servidor, del control lógico de los procesos empresariales, en dos interfaces independientes:
  • SAF "Sistema Administrativo Forseti", permite entre otras cosas, crear empresas, generar certificados de seguridad, respaldos, actualizar el sistema, etc.

  • CEF "Centro Empresarial Forseti", utilizado por los empleados de tu empresa, es la interfaz para el manejo de los distintos departamentos como contabilidad, facturación, etc.
