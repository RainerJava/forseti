Motores del proyecto Forseti
Menús principales
Los paneles izquierdo, inferior y derecho de este módulo aparecen vacíos porque no se utilizan.

Las columnas por defecto que muestra el panel de cuerpo principal de este módulo son la siguientes:

  • Clave. Es la clave del menú principal.
  • Descripción. Es la descripción del menú, y aparecerá en cada página de ayuda.

Procesos de los Menús principales

En los menús principales intervienen los procesos primarios de agregado, cambios, eliminación de menús.

Agregar elemento.

Permite dar de alta un nuevo elemento de menú principal. Este elemento se anexará a la barra de menús principales de la ayuda durante la generación, se irán agregando según su orden ascendente en la clave y nunca según su orden por descripción. El formulario para Agregar un nuevo elemento posee los siguientes objetos:
  • Clave. Indica aquí la clave que tendrá el menú principal.
  • Descripción. El nombre o una descripción breve la cual aparecerá en la barra de menús para distinguir este objeto.

Cambiar elemento.

Permite cambiar la clave o descripción del menú seleccionado de la lista. La clave solo se puede cambiar en caso de que no existan elementos de menús secundarios dependientes.

Eliminar elemento.

Permite eliminar el elemento de menú principal seleccionado de la lista. Solo se puede eliminar en caso de que no existan elementos de menús secundarios dependientes.
Errores comunes en este módulo


No se puede cambiar o eliminar este registro porque tiene registros hijos asociados a el, Primero se tienen que cambiar o eliminar los hijos para poder cambiar o eliminar el registro. Este es un error muy común y sucede cuando se intenta cambiar o eliminar un menú principal que tiene dependencias de menús secundarios. Según la integridad referencial de bases de datos, eliminar un objeto que tiene objetos que dependen de el, debe eliminar también estas dependencias para que no queden huérfanas, por lo tanto, primero debemos asignarle otro padre a nuestros menús dependientes, y después eliminar o cambiar.
