Motores del proyecto Forseti
Barra de información
El panel de la barra de información, nos muestra la fecha, y la hora de la máquina local ( no del Servidor Forseti ).

Es también en esta barra que se muestra información sobre: Versión, Servidor, Host, Usuario o Cliente y Compañía, esta última solo representada en el CEF y REF ya que el SAF no se conecta a alguna base de datos (empresa) en especial.

Version: Es la versión del sistema forseti que maneja el servidor al que se está conectado. Esta cambia automáticamente cuando se actualiza el servidor Forseti a una nueva versión.

Servidor: Muestra el nombre del servidor proveedor del sistema Forseti al que se está conectado. Los nombres de los servidores Forseti no son los nombres de dominio del servidor, son nombres asignados a gusto del dueño del servidor Forseti y solo visibles para Forseti.

Host: Es el nombre del PC o la dirección IP del PC conectado a Forseti. El host se utiliza para registros de sesion en el servidor Forseti y la seguridad que esto implica para resolver un eventual ataque al servidor.

Compañía: Al registrarnos, este campo nos muestra la base de datos a la que estamos conectados. solo aplica al CEF y REF

Usuario o Cliente: Al igual que la compañía, nos muestra el usuario del CEF o SAF, o el cliente de la empresa si es por el REF, con el que nos conectamos.
