Crear un live DVD desde windows o linux

Cómo grabar un DVD en Windows

Para instalar Lubuntu en un equipo que actualmente funciona con Windows, es necesario descargar el archivo de instalación y grabarlo en el DVD, que luego se puede utilizar para instalar Lubuntu. Una vez que hayas descargado el archivo, sigue las siguientes instrucciones para grabar tu DVD, según la versión de Windows que estás usando actualmente.

En Windows 7/8

  1. Haz Click derecho en la imagen ISO y selecciona "Open with > Windows Disc Image Burner".

  2. Selecciona la unidad de discos (disc burner) y presiona "Burn". Si marcas "Verify disc after burning", se confirmará que la imagen ISO se ha quemado correctamente.
En Windows 95/98 / ME / 2000 / XP / Server 2003 / Vista7
  1. Descarga e instala "Infra Recorder", este es un programa de grabación de imágenes, es software libre.

  2. Inserta un DVD en blanco en la unidad y selecciona "Do Nothing" o "Cancel" si el cuadro de diálogo de ejecución automática aparece.

  3. Abre Infra Recorder y haz clic en el botón "Write Image" en la pantalla principal. Alternativamente, puedes seleccionar el menú "Actions", después "Burn Image".

  4. Selecciona el archivo de imagen ISO y, a continuación, haz clic en "Open".

  5. En el cuadro de diálogo, haz clic en "Ok". 


Como grabar un DVD en Ubuntu

  1. Este procedimiento puede variar significativamente dependiendo de la versión que se estás utilizando.

  2. Inserta un DVD en blanco en la unidad. Una ventana de "CD/DVD Creator" o "Choose Disc Type" puede aparecer. Cierra, ya que no la vamos a utilizar.

  3. Busca la imagen ISO que descargaste, haz click derecho en esta, y selecciona "Write to Disc".

  4. Cuando pregunte "Write disc to", debes tener opciones como "File image" así como también tu CD drive. Selecciona tu CD drive. Tu CD drive puede parecer algo como "BD-MLT UJ-210S".

  5. Selecciona la velocidad de escritura. La opción por defecto debe dar buenos resultados.

  6. Inicializa el proceso de quemado.

  7. Después de que este se haya terminado, verifica que tu DVD contenga múltiples archivos y carpetas, y no solo el archivo ISO. De esta manera estarás seguro de que el proceso se ha generado con éxito.

  8. Si por alguna razón aplicación estándar no funciona, intenta hacerlo con la aplicación Gnomebaker.
