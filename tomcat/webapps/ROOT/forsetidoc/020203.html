Instalando el servidor forseti
Para la instalación del servidor web forseti, podemos utilizar el mismo servidor físico en donde instalamos PostgreSQL o uno diferente. Forseti es una aplicación web lo que significa que utiliza un servicio web para ser accesada. Forseti esta basado en la tecnología Servlet 2.0 y JSP 1.2, ambas son tecnologías de programas Java que se ejecutan en un servidor web, y construyen páginas web a partir de datos dinámicos. Apache Tomcat es el contenedor de servlets y jsp, es decir, es el módulo que utiliza forseti para ejecutar los servlets y páginas jsp, que permiten interactuar con el motor PostgreSQL y el navegador del usuario. Además, forseti se apoya en la tecnología SSL para construir certificados y llaves para que toda la información pueda viajar cifrada, para esto, utiliza la aplicación OpenSSL. Por ultimo, pero de suma importancia, forseti utiliza una aplicación llamada RSYNC para poder soportar tareas de respaldo y otras tareas de almacenamiento de datos alternas al servicio de base de datos que brinda PostgreSQL.

En resumen, los siguientes elementos se deben instalar para que el servdor forseti pueda funcionar: "Java OpenJDK 6", "OpenSSL", "RSYNC" y "Forseti soportado en Apache Tomcat". Empezaremos por la instalación del servicio de OpenJDK 6.

$ sudo apt-get install openjdk-6-jre

Si todo va bien, se instalará la maquina virtual de java y sus herramientas y podemos seguir con nuestro siguiente servicio que va a ser OpenSSL.

$ sudo apt-get install openssl

Ya que OpenSSL se utiliza para muchos procesos del sistema, lo mas probable es que ya se encuentre instalado. De todas formas, se pudiera verificar antes de la instalación de la siguiente manera:

$ dpkg -s openssl

Ahora continuamos con el servicio RSYNC

$ sudo apt-get install rsync

Por ultimo, vamos a instalar la aplicación web forseti soportada por tomcat. este se instala de una manera distinta a los demás, primero la descargamos con el comando wget, luego movemos al directorio de instalación y desempaquetamos el archivo comprimido:

$ wget http://www.forseti.org.mx/descargas/forseti.tar.gz
$ sudo mv forseti.tar.gz /usr/local
$ cd /usr/local
$ sudo tar zxf forseti.tar.gz

Hasta el momento, obtenemos el servidor forseti bajo el directorio /usr/local/forseti. Este directorio y todo su contenido hay que protegerlo para que pertenezca al super usuario y no pueda ser accedido desde ningun otro lugar. Para esto usaremos el comando chown, chgrp y chmod de la siguiente manera:

$ sudo chown -R root forseti
$ sudo chgrp -R root forseti
$ sudo chmod -R 755 forseti

Por ultimo, debemos borrar el archivo comprimido puesto en /usr/local

$ sudo rm forseti.tar.gz

Si todo salió bien, ya tenemos todos los elementos instalados para poder empezar a configurar nuestro servidor forseti.
